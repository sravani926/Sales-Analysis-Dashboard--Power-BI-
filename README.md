## 📝 Description
This project presents an interactive **Sales Analysis Dashboard** built using Power BI. It helps analyze business performance by tracking key metrics such as total sales, profit, and customer trends. The dashboard enables users to gain actionable insights through dynamic visualizations and filters.

---

## 🚀 Features
- 📌 KPI cards for Total Sales and Total Profit  
- 📅 Year and Quarter slicers for dynamic filtering  
- 🌍 Sales analysis by State and Category  
- 📈 Monthly and yearly sales trends  
- 🧩 Category-wise performance breakdown  
- 👥 Customer-level detailed data view  

---

## 🛠 Tech Stack
- **Visualization Tool:** Power BI  
- **Data Processing:** Power Query  
- **Data Source:** Excel / CSV Dataset  

---

## 📂 Installation
1. Download the `.pbix` file from this repository  
2. Open using **Power BI Desktop**  
3. Connect dataset if required  
4. Click **Refresh** to load data  

---

## 📊 Usage
- Use slicers to filter by Year and Quarter  
- Monitor KPIs to evaluate performance  
- Analyze charts for trends and insights  
- Explore customer data for detailed analysis
